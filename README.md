# EVENT MANAGEMENT PROJECT WEB APP

Event Management Web App made as practice project to learn deeper about React and React Router V7 as well learn and implement new things like Typescript. Event Management Web App contains:

- CRUD Event
- Authentication
- JWT and Refresh Token with Http-only-cookies
- Protected routes
- Responsive UI
- Reuseable Components
- REST API Integration
- Validation Input
- Optimize route
- CORS handling

## REQUIREMENT

In order to start this project localy inside your local machine, some requirement needed.

- Installed NodeJs

## GET STARTED

1. Clone this repository to your local working directory
2. Open a new ternimal and run `cd backend` to move current directory into backend directory
3. run `npm install` to install all the depedencies needed by the backend
4. run `npm start` to start development server
5. Open a new terminal while keep backend server live and run `cd frontend` to move current directory into frontend directory
6. run `npm install` to install all the depedencies needed by the frontend
7. run `npm run dev` or `npm start` to start frontend
8. Event Management App served

### !IMPORTANT

Make sure to always keep backend and frontend terminal server live.
